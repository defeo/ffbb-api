# FFBB api

Scraper for www.ffbb.com site.

## Usage

### Installation

~~~
npm install
~~~

### Getting help

~~~
> ./ffbb -h
~~~

### Getting a license number

~~~
> ./ffbb license -h
~~~

~~~
> ./ffbb license -n Parker -p Tony 
1 licencié trouvé
